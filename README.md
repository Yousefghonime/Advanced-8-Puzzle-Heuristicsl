
# 8-Puzzle AI Solver (Tkinter + BFS/DFS/UCS/A*)

Modular structure separating algorithms, core logic, and UI.

## Run
```bash
python3 main.py
```

## Structure
- `core/`: PuzzleState & utilities
- `algorithms/`: BFS, DFS, UCS, A*
- `ui/`: Tkinter GUI
- `main.py`: entry point

## Requirements
- Python 3.8+
- Tkinter (included with most Python installations)
